# O'quv Markazi Boshqaruv Tizimi

Bu loyiha o'quv markazlari uchun yaratilgan zamonaviy va samarali boshqaruv tizimi hisoblanadi. Ushbu platforma orqali o'quvchilar, o'qituvchilar, davomat va baholarni boshqarish imkoniyati mavjud.

## Asosiy Xususiyatlar

- **Foydalanuvchi boshqaruvi** - O'quvchilar va o'qituvchilarni ro'yxatdan o'tkazish va boshqarish
- **Davomat nazorati** - Darslarga qatnashuvlarni kuzatish va hisobot tayyorlash
- **Baholash tizimi** - O'quvchilarning natijalarini baholash va tahlil qilish
- **Kurslar boshqaruvi** - O'quv dasturlari va guruhlarni boshqarish
- **Interfeys** - Zamonaviy va qulay foydalanuvchi interfezi

## Texnologiyalar

- HTML5, CSS3, JavaScript (ES6+)
- Font Awesome ikonalari
- Mobil qurilmalar uchun moslashuvchan dizayn (Responsive Design)
- Tez ishlash uchun optimallashtirilgan kod
